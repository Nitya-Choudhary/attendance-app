# Smart Classroom Manager - Quick Start Guide

## 🚀 Getting Started in 5 Minutes

### 1. Access the Application

**Live Demo:** https://3000-is29kb11xu2panqobjxa5-86eeb915.sg1.manus.computer

### 2. Login

Click "Sign In with Manus" and authenticate with your Manus account.

### 3. Choose Your Role

**For Teachers (Admin):**
- Create assignments with deadlines
- Mark student attendance
- View and reply to student feedback
- Export attendance reports as CSV

**For Students:**
- View your attendance records and percentage
- Submit assignments (text or file upload)
- Send feedback to your teacher
- Track assignment deadlines

## 📊 Key Features

### Dashboard
- **Summary Cards:** Quick overview of key metrics
- **Recent Activity:** Latest assignments and feedback
- **Role-Specific Views:** Different dashboards for teachers and students

### Attendance
- **Mark Attendance:** Teachers can mark students present/absent
- **View Records:** See attendance history by subject and date
- **Calculate Percentage:** Automatic attendance percentage calculation
- **Export CSV:** Download attendance reports

### Assignments
- **Create:** Teachers add assignments with title, description, deadline
- **Submit:** Students submit via text or file upload
- **Track:** See submission status and deadlines
- **Download:** Teachers can download student submissions

### Feedback
- **Submit:** Students send feedback/suggestions to teachers
- **Reply:** Teachers respond to student feedback
- **Track:** See feedback status (pending/replied)
- **Search:** Find feedback by date or student

## 🆘 Troubleshooting

### Can't Login?
- Clear browser cookies and try again
- Check your Manus account is active
- Try a different browser

### File Upload Not Working?
- Check file size (max 10MB)
- Supported formats: PDF, DOC, DOCX, TXT, JPG, PNG
- Ensure you have stable internet connection

### Attendance Not Showing?
- Refresh the page (Ctrl+R or Cmd+R)
- Make sure you're logged in as a teacher
- Check the date and subject filters

## 🎯 Next Steps

1. **Create Your First Assignment** (Teachers)
   - Go to Assignments → Create New Assignment
   - Fill in title, description, subject, and deadline
   - Click "Create Assignment"

2. **Mark Attendance** (Teachers)
   - Go to Attendance → Mark Attendance
   - Select date, subject, and student
   - Choose Present or Absent
   - Click "Mark Attendance"

3. **Submit Assignment** (Students)
   - Go to Assignments
   - Click "Submit Assignment"
   - Enter text or upload file
   - Click "Submit"

4. **Send Feedback** (Students)
   - Go to Feedback → Submit Feedback
   - Enter subject and message
   - Click "Submit Feedback"
